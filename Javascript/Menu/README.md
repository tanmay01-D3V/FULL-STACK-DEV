# Nagori Caffe Menu

A dynamic menu application for "Nagori Caffe" that allows users to select beverages and see their prices. This project demonstrates interactive DOM manipulation using JavaScript.

## Features

-   **Beverage Selection**: Users can select from a variety of beverages (Tea, Ice Tea, Green Tea, Latte, Espresso).
-   **Dynamic Pricing**: The application displays the price for each item.
-   **Interactive Interface**: Clean and simple user interface for easy selection.

## Technologies Used

-   **HTML5**: structured the menu layout.
-   **CSS3**: Styled the menu cards and layout, using flexbox/grid and custom fonts (Inter).
-   **JavaScript**: Handles user interactions and form processing (logic implementation pending/in-progress).

## Getting Started

1.  Clone the repository.
2.  Open `index.html` in a web browser.
3.  Select items from the menu.
