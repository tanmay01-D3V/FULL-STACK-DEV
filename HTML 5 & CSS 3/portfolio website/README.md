# Portfolio Website

A multi-page personal portfolio website showcasing skills, services, and contact information.

## Contents
- `Portfolio.html`: Main landing page of the portfolio.
- `About.html`: Section detailing personal information and background.
- `services.html`: Overview of services offered.
- `contact.html`: Contact form and details.
- `Assets/`: Directory containing project assets (images, icons, etc.).
