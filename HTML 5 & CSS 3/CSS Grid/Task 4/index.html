What does auto do in grid rows?

auto adjusts row height based on content.

Which row grows based on content?

The row with auto value.

Can rows have different heights?

Yes, each row can have a different height.

What happens if content is very large?

The row expands to fit the content (if auto).

Where are grid rows useful in real layouts?

For:
  • Headers
  • Content sections
  • Footers
