Where does gap add space?

Between grid items, not outside them.

How is gap different from margin?

gap adds space between items only margin adds space around elements individually

Does gap affect container size?

No, it only affects spacing inside the container.

Can we use gap only for rows or columns?

Yes:
  1. row-gap
  2. column-gap
  3. column-gap

Why is gap preferred in Grid layouts?

It keeps spacing clean, uniform, and easy to manage.
