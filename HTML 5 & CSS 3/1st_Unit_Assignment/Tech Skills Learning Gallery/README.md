# Tech Skills Learning Gallery

> "Architecting the Future, One Component at a Time."

This project is an interactive gallery showcasing various technical skills and technologies, built to demonstrate the integration of different layers of a full-stack application.

## Features

- **Interactive Gallery**: visually appealing cards representing different technologies.
- **Filtering System**: Filter skills by category:
    - Frontend
    - Backend
    - Problem Solving
- **Detailed Tech Stack Explanations**: Insights into why specific technologies were chosen:
    - **MERN Stack**: The Application Layer for dynamic content.
    - **C++ & DSA**: The Logic Layer for efficiency and optimization.
    - **HTML & CSS**: The Presentation Layer for professional and responsive design.

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Frameworks/Libraries**: React.js, Tailwind CSS, GSAP, Next.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Tools**: Postman

## Project Structure

The project highlights the transition from static HTML/CSS sites to dynamic applications powered by a backend stack, enabling features like database storage for skill details and automatic content rendering.
