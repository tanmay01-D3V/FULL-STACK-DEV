# Fun Nature Gallery

This project is a responsive image gallery showcasing beautiful nature photos. It is designed as a learning exercise for CSS Grid layout techniques.

## Features

-   **CSS Grid Layout**: Utilizes CSS Grid for a responsive and flexible gallery layout.
-   **Sidebar Filters**: Includes a sidebar with checkbox filters for different nature categories (Landscape, Forest, Mountain, Water), mimicking a real-world gallery application.
-   **Interactive Elements**: Features hover effects on gallery items.
-   **Responsive Design**: The gallery adjusts to different screen sizes.

## Technologies Used

-   **HTML5**: For semantic structure.
-   **CSS3**: For styling, specifically focusing on CSS Grid properties like `grid-template-columns`, `grid-gap`, and alignment properties.

## Getting Started

1.  Clone the repository.
2.  Open `index.html` in your web browser to view the gallery.
