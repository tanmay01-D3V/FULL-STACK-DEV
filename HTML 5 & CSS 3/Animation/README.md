# Animation Projects

This folder contains various CSS animation examples and assignments.

## Contents

- **Emoji Animation**: `Emoji.html`, `EmojiStyle.css`
- **House Animation**: `House.html`, `HouseStyle.css`
- **Traffic Signal Animation**: `Traffic.html`, `TrafficStyle.css`
- **Playground**: `playground.html` - A general playground for testing animations.

## Description

These files demonstrate different CSS animation techniques, including keyframes, transitions, and transforms.
