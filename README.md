## Portfolio Project

A personal portfolio website built using HTML5 and CSS3.

### Structure
The portfolio is located in `HTML 5 & CSS 3/portfolio/` and consists of the following pages:
- **Portfolio.html**: The main landing page.
- **About.html**: Information about me.
- **services.html**: Details about the services I offer.
- **contact.html**: A contact form and information.

- Responsive design using CSS3.
- Navigation between different sections.
- Clean and modern UI.
